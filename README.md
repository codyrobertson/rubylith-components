# rubylith-components
Contract-Driven Component Registry for UI components with execution contracts and environment capabilities

## Test Infrastructure

This project includes a robust test infrastructure with proper integration between `ApiTestHelper` and `TestDatabase` classes for reliable test execution.

### Quick Start

```bash
# Install dependencies
npm install

# Run tests
npm test

# Build the project
npm run build
```

### Test Infrastructure Features

- **TestDatabase**: Singleton class for managing test user data
- **ApiTestHelper**: Integrated helper for API testing with authentication
- **User Fixtures**: Predefined test user data
- **Integration Tests**: Comprehensive test suite demonstrating proper usage

See `tests/README.md` for detailed documentation on the test infrastructure.
